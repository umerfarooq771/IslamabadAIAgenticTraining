# 🤖 EX2-FirstAgent Challenges

Welcome to the **AI Agent Development** challenges! These exercises will help you learn how to build sophisticated AI agents using Azure AI Foundry and modern web interfaces.

## 🏗️ **What You'll Build**

Transform basic agent examples into specialized, domain-specific AI assistants with:
- **Custom agent instructions** for specific use cases
- **Session management** and conversation memory
- **Modern web interfaces** with Chainlit
- **Enhanced user experience** with personalized interactions

## 📚 **Available Challenges**

### 🌍 [**Travel Companion Challenge**](./ex2-travel-companion-challenge.md)
**Time:** 15 minutes | **Difficulty:** Beginner  
**Goal:** Create a specialized travel advisor agent with destination memory and personalized recommendations.

**What you'll learn:**
- Domain-specific agent customization
- Session state management with agents
- Travel-themed UI/UX design
- Context-aware conversation handling

## 🎯 **Learning Objectives**

By completing these challenges, you will understand:

1. **Agent Specialization** - How to customize AI agents for specific domains
2. **Conversation Management** - Using Azure AI Foundry threads for context
3. **Web Integration** - Combining agents with modern chat interfaces  
4. **User Experience** - Creating engaging, personalized AI interactions
5. **Session Persistence** - Maintaining state across conversations

## 🚀 **Getting Started**

1. **📁 Review the base examples** in the `samples/` folder first
2. **🎯 Choose a challenge** that matches your skill level
3. **⏱️ Set aside 15-20 minutes** for focused development
4. **💡 Start with basic requirements** then add bonus features
5. **🧪 Test thoroughly** to ensure everything works

## 🔧 **Prerequisites**

- Complete setup from the main EX2 README
- Azure AI Foundry credentials configured
- Chainlit installed and working
- Basic understanding of Python and async programming
